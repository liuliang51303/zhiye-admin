初始模板这是每个页面最基本的结构配置, 参阅主体布局和组件来布置主体容器里的内容

下载模板 (暂无)
  • head, header, aside, copyright这几个部分建议以引用的方式, 方便统一修改
  • <header></header>顶部, <aside></aside>左侧边栏, <main></main>主体
  • .MT-header-bar是在移动端显示的导航切换按钮, 不可移除
  • 有data-pjax属性的链接是以pjax的方式无刷新替换<main>部分, 所以每个页面的独立功能要写在<main>里面的page script位置, 主体容器里的链接也建议使用这种方式
  • 页面独立插件在page script位置以$.getScript的方式加载执行
                            
                                <!DOCTYPE html>
                                <html>
                                <head>
                                    <meta charset="utf-8">
                                    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
                                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                                    <meta name="format-detection" content="telephone=no">
                                    <title>页面标题</title>
                                    <link rel="icon" href="favicon.ico">
                                    <!-- CSS icofont -->
                                    <link rel="stylesheet" href="icofont/style.css">
                                    <!-- CSS global -->
                                    <link rel="stylesheet" href="css/style.min.css">
                                    <!-- jQuery -->
                                    <script src="js/jquery-1.9.1.min.js"></script>
                                </head>
                                <body>
                                    <!-- header start -->
                                    <header>
                                        <!-- brand start -->
                                        <div class="brand">
                                            <h4 class="title">项目名称</h4>
                                            <p>系统名称</p>
                                        </div>
                                        <!-- brand start -->
                                        <!-- MT bar start -->
                                        <div class="MT-header-bar">
                                            <button class="MT-bar-toggler collapsed" type="button" data-toggle="collapse" data-target="aside">
                                                <i class="icon-list"></i>
                                            </button>
                                        </div>
                                        <!-- MT bar over -->
                                    </header>
                                    <!-- header over -->
                                    <!-- aside start -->
                                    <aside>
                                        <!-- nav start -->
                                        <div class="side-nav">
                                            <nav class="nav">
                                                <!-- item start -->
                                                <div class="item">
                                                    <a class="active" href="#" data-pjax><i class="icon-home"></i><span>首页</span></a>
                                                </div>
                                                <!-- item over -->
                                                <!-- item start -->
                                                <div class="item">
                                                    <a><i class="icon-gear"></i><span>一级导航</span></a>
                                                    <div class="nav-sub">
                                                        <div class="item"><a href="#" data-pjax>二级导航</a></div>
                                                        <div class="item"><a href="#" data-pjax>二级导航</a></div>
                                                    </div>
                                                </div>
                                                <!-- item over -->
                                            </nav>
                                        </div>
                                        <!-- nav over -->
                                    </aside>
                                    <!-- aside over -->
                                    <!-- main start -->
                                    <main>
                                        <!-- banner start -->
                                        <div class="banner">
                                            <div class="banner-title">
                                                <h2>页面标题<small>副标题</small></h2>
                                            </div>
                                            <div class="banner-action">
                                                <!-- banner功能区 -->
                                            </div>
                                        </div>
                                        <!-- banner over -->
                                        <!-- container start -->
                                        <div class="container-fluid">
                                            <!-- 主体容器, 放入网格 -->
                                        </div>
                                        <!-- container over -->
                                        <!-- page script start -->
                                        <script type="text/javascript">
                                        <!-- page script -->
                                        </script>
                                        <!-- page script over -->
                                        <!-- copyright start -->
                                        <div class="copyright">
                                            <p>Powered by <a href="http://www.maszy.cn" target="_blank">© ZhiYe Software Technology Co.,Ltd</a></p>
                                        </div>
                                        <!-- copyright over -->
                                    </main>
                                    <!-- main over -->
                                    <!-- JS plugins -->
                                    <script src="js/plugins.js"></script>
                                    <!-- JS global -->
                                    <script src="js/scripts.js"></script>
                                </body>
                                </html>

    js/scripts.js是前端全局js, 不要编辑修改

    js/plugins.js打包了以下插件供全局使用

  • Bootstrap (bootstrap的全部功能)
  • Nprogress (页面加载进度条)
  • jQuery.pjax (页面无刷新替换)
  • Waves (按钮点击反馈)
  • jQuery.mCustomScrollbar (滚动条美化)
  • HighCharts (图表, 包括了highcharts-more.js增强包和modules/exporting.js导出功能)
  • Tabbedcontent (选项卡)
  • jQueryValidatio (表单验证)
  • DataTables (数据表格)
  • Laydate (日期和时间控件)
  • Chosen (下拉列表美化和功能增强)
  • Toastr (toast提示框)
  • Dropzone (多文件上传)
  • Hotkeys (快捷键)
  • Clipboard (内容复制)
  • Prism (代码美化)
