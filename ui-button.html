按钮button, a, input / 推荐本页面功能用button, 页面跳转用a

按钮类型三种不同类型的按钮, 样式已统一, 按需使用

a a + icon
  • 首先选择并添加一个按钮类型, 然后在这个按钮上追加示例右侧的类名或属性, 按需改变它的大小, 颜色等
  • 按钮组件的初始类名都是.button
  • 按钮内加图标, 需要把文字部分用<span></span>嵌套
  • 当一个区域内有多个按钮且排列比较拥挤混乱时, 试着将它们嵌套在.button-group里面
  • 推荐使用button和a, 可相互转换标签, 方便添加和更换图标, 带有点击反馈
